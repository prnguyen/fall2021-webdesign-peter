Free Resources to learn HTMl, CSS, and JavaScript

Eloquent JavaScript

This is the free online version of the popular book you can use to learn modern JavaScript, Eloquent JavaScript written by Marijn Haverbeke. This book is widedly endorsed and backed by many companies and coding bootcamps such as Mozilla, Hack Reactor, and Holberton School, to name a few. This book is also available in other languages like Persian, Spanish, Portuguese, Arabic, Bulgarian, and Russian.

JavaScript: The Definitive Guide - Master the World's Most-Used Programming Language

This is a great book I found from San Jose Public Library that covers Javscript very comprehensively. It's a popular and highly rated title on Amazon to get started with Javascript. It's written by David Flanagan, an experienced programmer and author of many popular programming books about Java and Javascript with the O'Reilly publisher.

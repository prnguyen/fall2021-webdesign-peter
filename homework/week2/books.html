This is the free online version of the popular book you can use to learn modern JavaScript, Eloquent JavaScript written by Marijn Haverbeke. This book is widedly endorsed and backed by many companies and coding bootcamps such as Mozilla, Hack Reactor, and Holberton School, to name a few. This book is also available in other languages like Persian, Spanish, Portuguese, Arabic, Bulgarian, and Russian.
